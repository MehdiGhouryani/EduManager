# 🎓 EduManager – سیستم مدیریت آموزش (LMS)

[![Python](https://img.shields.io/badge/Python-3.13-blue.svg)](https://www.python.org/)

> (◡‿◡✿) خوش آمدید!  
> **EduManager** یک پلتفرم مدیریت یادگیری مبتنی بر جنگو است که به اساتید و دانشجویان امکان مدیریت دوره‌ها، محتوا، تکالیف، آزمون‌ها و اعلان‌ها را می‌دهد. تمرکز پروژه بر **بک‌اند قوی** و **فرانت‌اند ساده و مدرن** است.

## ✨ ویژگی‌های اصلی (≧◡≦)

- مدیریت کاربران و نقش‌ها (ادمین، استاد، دانشجو) با احراز هویت کامل (ورود، ثبت‌نام، بازیابی رمز، «مرا به خاطر بسپار»)
- مدیریت دوره‌ها – ایجاد، ویرایش، حذف دوره با ظرفیت، تاریخ، استاد مرتبط
- مدیریت محتوا – آپلود فایل‌های PDF، ویدئو، متن توسط استاد در پنل اختصاصی
- سیستم تکالیف – تعریف تکلیف با ددلاین، ارسال فایل توسط دانشجو، نمره‌دهی و بازخورد استاد
- سیستم آزمون آنلاین – ساخت آزمون چهارگزینه‌ای، شرکت دانشجو، تصحیح خودکار و نمایش نمره
- اعلان‌های داخلی – دریافت اعلان برای رویدادهای مهم (نمره‌دهی، ثبت‌نام) به همراه منوی زنگوله
- تابلوی اعلانات عمومی – نمایش پیام‌های سراسری در صفحات لاگین و داشبورد
- پروفایل کاربری – ویرایش اطلاعات شخصی، تغییر رمز عبور
- داشبورد تحلیلی – نمایش آمار پیشرفت دانشجو و آمار استاد
- رابط کاربری مدرن و واکنش‌گرا (RTL) با Bootstrap 5

## 🚀 نصب و راه‌اندازی

### پیش‌نیازها
- Python 3.10 یا بالاتر
- pip

### مراحل

1. **کلون مخزن**
   ```bash
   git clone https://github.com/yourusername/EduManager.git
   cd EduManager
   ```

2. **ایجاد و فعال‌سازی محیط مجازی**
   ```bash
   python -m venv venv
   source venv/bin/activate      # Linux/Mac
   venv\Scripts\activate         # Windows
   ```

3. **نصب وابستگی‌ها**
   ```bash
   pip install -r requirements.txt
   ```

4. **اعمال migrations**
   ```bash
   python manage.py migrate
   ```

5. **ایجاد کاربر ادمین**
   ```bash
   python manage.py createsuperuser
   ```

6. **اجرای سرور توسعه**
   ```bash
   python manage.py runserver
   ```

7. دسترسی به پروژه در آدرس `http://127.0.0.1:8000`

## 📌 نحوه استفاده

### دانشجو
- ثبت‌نام و ورود
- مشاهده لیست دوره‌های فعال و ثبت‌نام در آنها
- مشاهده محتواهای دوره (PDF، ویدئو، متن)
- ارسال تکلیف و مشاهده نمرات
- شرکت در آزمون‌های چهارگزینه‌ای
- مشاهده اعلان‌ها و ویرایش پروفایل

### استاد
- ورود به پنل استاد (`/instructors/dashboard/`)
- ایجاد دوره جدید
- مدیریت محتوا (افزودن، ویرایش، حذف فایل‌ها)
- مدیریت تکالیف و نمره‌دهی
- مدیریت آزمون‌ها و افزودن سوالات
- مشاهده لیست دانشجویان هر دوره

### ادمین
- دسترسی کامل به پنل ادمین Django (`/admin`)
- مدیریت کاربران و نقش‌ها
- مدیریت همه دوره‌ها، محتواها، تکالیف، آزمون‌ها
- ارسال اعلان همگانی از طریق اکشن ادمین
