## ✨ ویژگی‌های اصلی (≧◡≦)

- **مدیریت کاربران و نقش‌ها:** (ادمین، استاد، دانشجو) با احراز هویت کامل (ورود، ثبت‌نام، بازیابی رمز، «مرا به خاطر بسپار»).
- **مدیریت دوره‌ها:** ایجاد، ویرایش و حذف دوره با تعیین ظرفیت، تاریخ و استاد مرتبط.
- **مدیریت محتوا:** آپلود فایل‌های PDF، ویدئو و متن توسط استاد در پنل اختصاصی.
- **سیستم تکالیف:** تعریف تکلیف با ددلاین، ارسال فایل توسط دانشجو، نمره‌دهی و بازخورد استاد.
- **سیستم آزمون آنلاین:** ساخت آزمون چهارگزینه‌ای، شرکت دانشجو، تصحیح خودکار و نمایش نمره.
- **سیستم اعلان‌ها:** - اعلان‌های داخلی: دریافت اعلان برای رویدادهای مهم (نمره‌دهی، ثبت‌نام) به همراه منوی زنگوله.
  - تابلوی اعلانات عمومی: نمایش پیام‌های سراسری در صفحات لاگین و داشبورد.
- **پروفایل کاربری:** امکان ویرایش اطلاعات شخصی و تغییر رمز عبور.
- **داشبورد تحلیلی:** نمایش آمار پیشرفت دانشجو و آمار استاد.
- **رابط کاربری:** مدرن و واکنش‌گرا (RTL) با استفاده از Bootstrap 5.

## 🚀 نصب و راه‌اندازی

### پیش‌نیازها
- Python 3.10 یا بالاتر
- `pip` (مدیریت پکیج پایتون)

### مراحل اجرا (گام به گام)

**۱. کلون کردن مخزن**
```bash
git clone [https://github.com/yourusername/EduManager.git](https://github.com/yourusername/EduManager.git)
cd EduManager
```

**۲. ایجاد و فعال‌سازی محیط مجازی (Virtual Environment)**
```bash
python -m venv venv

# برای سیستم‌عامل لینوکس و مک:
source venv/bin/activate

# برای سیستم‌عامل ویندوز:
venv\Scripts\activate
```

**۳. نصب وابستگی‌ها**
```bash
pip install -r requirements.txt
```

**۴. اعمال migrations به پایگاه داده**
```bash
python manage.py migrate
```

**۵. ایجاد کاربر ادمین (Superuser)**
```bash
python manage.py createsuperuser
```

**۶. اجرای سرور توسعه**
```bash
python manage.py runserver
```

**۷. دسترسی به سیستم**
مرورگر خود را باز کرده و آدرس `http://127.0.0.1:8000` را وارد کنید.

## 🧪 ساختار پروژه

پروژه به صورت ماژولار و ساختاریافته طراحی شده است:

```text
EduManager/
├── academy/           # دوره‌ها، دانشجویان، محتوا، آزمون‌ها
├── accounts/          # پروفایل کاربران و نقش‌ها
├── instructors/       # پنل اختصاصی استاد (دوره، محتوا، تکالیف، آزمون)
├── assignments/       # مدل‌ها و ویوهای تکالیف و ارسال‌های دانشجویان
├── notifications/     # سیستم اعلان‌های داخلی کاربران
├── announcements/     # اعلانات عمومی سیستم
├── config/            # تنظیمات اصلی پروژه (settings, urls)
├── templates/         # قالب‌های HTML هماهنگ با Bootstrap RTL
├── media/             # فایل‌های آپلودی کاربران و مدیاها
├── requirements.txt   # لیست پکیج‌های مورد نیاز
└── manage.py          # اسکریپت اصلی جنگو
```

## 📌 نحوه استفاده

### 👨‍🎓 دانشجو
- ثبت‌نام و ورود به پنل کاربری.
- مشاهده لیست دوره‌های فعال و ثبت‌نام در آن‌ها.
- دسترسی به محتواهای دوره (PDF، ویدئو، متن).
- ارسال فایل تکالیف پیش از پایان مهلت و مشاهده نمرات.
- شرکت در آزمون‌های چهارگزینه‌ای.
- مشاهده اعلان‌ها و ویرایش اطلاعات پروفایل.

### 👨‍🏫 استاد
- ورود به پنل اختصاصی استاد (`/instructors/dashboard/`).
- ایجاد دوره‌های آموزشی جدید.
- مدیریت محتوا (افزودن، ویرایش، حذف فایل‌ها).
- مدیریت تکالیف ارسالی، نمره‌دهی و ثبت بازخورد.
- مدیریت آزمون‌ها و افزودن سوالات تستی.
- مشاهده لیست دانشجویان ثبت‌نام‌شده در هر دوره.

### 👑 ادمین
- دسترسی کامل و همه‌جانبه به پنل ادمین جنگو (`/admin`).
- مدیریت تمامی کاربران و نقش‌ها.
- کنترل همه دوره‌ها، محتواها، تکالیف و آزمون‌ها.
- ارسال اعلان همگانی از طریق اکشن‌های پنل ادمین.
